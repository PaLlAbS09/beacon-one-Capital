Beacon One Capital – Landing Page
Project Overview

This project is a responsive landing page developed from a Figma design using modern frontend technologies. The website represents a real estate investment company called Beacon One Capital, showcasing property development services, projects, testimonials, and contact information.

The goal of this project is to convert a UI design into a fully responsive web page using HTML5, CSS3, Bootstrap, and Swiper.js.

Technologies Used

HTML5 – For semantic page structure

CSS3 – For custom styling and layout

Bootstrap 5 – For responsive grid system and components

Swiper.js – For testimonial slider functionality

Figma – Used as the design reference
